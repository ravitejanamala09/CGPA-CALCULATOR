# SGPA to CGPA Calculator

A simple web application that calculates CGPA from semester-wise SGPA values.

## Features

- Enter number of semesters
- Enter SGPA for each semester
- Automatic CGPA calculation
- Simple and beginner-friendly interface

## Technologies Used

- HTML
- JavaScript

## How to Run

1. Download the project
2. Open index.html in your browser
3. Click "Calculate CGPA"
4. Enter semester details
5. View the calculated CGPA
